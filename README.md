Основные компоненты (Классы и модули):\
FastAPI: Главный класс приложения, который управляет маршрутизацией и обработкой HTTP-запросов.\
joblib: Используется для десериализации (загрузки) предобученной модели из файла .joblib.\
pandas: Основной инструмент для обработки табличных данных. Внутри сервиса он преобразует входящий поток байтов в структуру DataFrame.\
StreamingResponse: Специальный класс FastAPI для передачи файлов. Он позволяет не хранить результат на диске сервера, а отдавать его «на лету» прямо в браузер пользователя.

Как запустить сервис:

Установка окружения:\
Откройте терминал в папке проекта и выполните:
python -m venv venv\
.\venv\Scripts\activate\
pip install fastapi uvicorn pandas python-multipart joblib scikit-learn imblearn

Запуск:\
Введите команду:\
uvicorn main:app --reload

Использование:\
Откройте браузер по адресу: http://127.0.0.1/docs \
Найдите метод POST /predict_json_file.\
Нажмите Try it out, выберите ваш CSV-файл и нажмите Execute.\
В ответе появится кнопка Download file — нажмите её, чтобы скачать результат.
